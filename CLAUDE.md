## Overview

This is a LaTeX resume built on the [Awesome CV](https://github.com/posquit0/Awesome-CV) template, compiled with XeLaTeX. The environment is managed with [devenv](https://devenv.sh) (Nix-based), providing a TeX Live installation and `tex-fmt` for formatting.

## Commands

```bash
tectonic resume.tex        # compile → resume.pdf (auto-fetches required packages)
tectonic cover_letter.tex  # compile → cover_letter.pdf
tex-fmt <file>.tex         # format a .tex file (runs automatically as a pre-commit hook)
```

The devenv shell (activated via `direnv allow`) provides Tectonic, and `tex-fmt`.

## Structure

- `resume.pdf` — the compiled output; generated by running `xelatex resume.tex`
- `resume.tex` — entry point; sets personal info, page layout, and `\input`s section files
- `resume/` — individual section files: `experience.tex`, `education.tex`, `projects.tex`, plus unused sections (`honors`, `summary`, etc.) that can be re-enabled in `resume.tex`
- `awesome-cv.cls` — the document class defining all CV macros (`\cventry`, `\cvskill`, etc.)
- `cover_letter.tex` — standalone cover letter (separate from the resume)
- `examples/` — upstream template examples; not part of the actual resume

## Key conventions

- The highlight color is `awesome-red`; change via `\colorlet{awesome}{...}` in `resume.tex`
- Sections are toggled by commenting/uncommenting `\input{resume/<section>.tex}` lines
- `tex-fmt --nowrap` runs automatically on every `git commit` via a devenv git hook
